# Smart funding ðŸ’°

Smart funding is a Web3 Blockchain Crowdfunding Platform where users can list campaigns with a target amount and a deadline using their web3 wallet and also fund the campaigns with ETH

## Folder structure

- [smart-contract](https://github.com/gurukishore111/Smart-Funding/tree/master/smart-contract 'smart-contract') - contains smart funding solidity code
- [client](https://github.com/gurukishore111/Smart-Funding/tree/master/client 'client') - contains frontend code

## Smart contract address

[0x757FFF7caAbb0e00241FC9a423F7dEc706d855C7](https://thirdweb.com/goerli/0x757FFF7caAbb0e00241FC9a423F7dEc706d855C7) - deployed on the Goreli network

## Running project

    npm install - install dependencies
    npm run dev - to start dev server
    npm run build - build the web app
